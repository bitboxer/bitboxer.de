  • OnlineTVRecorder

    Ich bin vorgestern auf etwas neues aufmerksam gemacht worden: eine Webseite namens OnlineTVRecorder. Das ist ein Video-Rekorder im Netz. Man sagt was man aufgenommen haben will, und er nimmt’s auf. Nach der vollendeten Aufnahmen kann man dann das File runterladen. Das runtergeladene File muß nur noch mit einem Tool dekodiert werden und schwups hat man dann die Aufnahme als WMV-Datei auf seiner Platte liegen. ( Eine Liste der unterstützen Sender ).
